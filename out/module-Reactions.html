<static> done(defaultDone, success, param)

Creates a 'done' function. A done function can be invoked only once, receives an error and callback in case of error invokes the defaultDone passing the error in case of success invokes success passing the data or the defaultDone if no success is present passing false as first argument and data as the second
Parameters:
Name Type Argument Description
defaultDone function the done function to delegate to.
success function <optional>
 the success function to delegate to.
param any <optional>
 a parameter to pass as third one to the callback function or second one to the success function. Intended for passing of additional argument(s) on success.
